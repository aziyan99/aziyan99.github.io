## README



Steps Recorder is a versatile Chrome extension designed to make documenting workflows easier than ever. With the ability to capture screenshots of the area around your cursor with each click and a built-in editor to add instructions, this tool is perfect for creating detailed tutorials, tracking processes, or recording user actions step by step.

## Key Features

-   **Smart Capture**: Automatically capture high-quality screenshots of the area around your cursor with every click.
-   **Step Editor**: Add step-by-step instructions directly to your screenshots.
-   **Flexible Step Management**:
    -   **Reorder Steps**: Easily move steps up or down to perfect the flow.
    -   **Text-Only Steps**: Add extra context with instruction-only steps (no image required).
    -   **Image Controls**: Zoom and pan within screenshots to highlight exactly what matters.
-   **Professional Export**: Export your guide to a clean, professionally styled PDF or print format.
-   **Privacy First**: All data is captured and stored locally on your device, ensuring your privacy is protected.

## Ideal For

-   **Tutorial Creators & Educators**: Craft clear, step-by-step guides for students or audiences.
-   **QA Engineers & Developers**: Document bugs, reproduction steps, or testing workflows with precision.
-   **Process Documentation**: Record internal processes and SOPs effortlessly.

## Note

To built the extension for production we need to run the `npm run dev` command before running the `build` command. Otherwise the built we outputing an error `Could not load CSS 'content_scripts/content-0.css'`.

```
npm run dev
npm run build
npm run start
```
