## CI-Base-Development-System

---

CI-Base-Development-System is a starter project or base, to start developing web app with Codeigniter framework. This starter include multiple user authorization access system, user management, auth system(login and register), menu management system and access management system to spesific user role.
![First Post](/images/first-post.png "First Post")
This web app was built with codeigniter 3.x.x framework and AdminLTE as a dashboard template, if you want to use this starter project please chech the authentication and authorization system because this starter was built a long long time ago :v [Goto repository](https://github.com/aziyan99/ci-base-development-system "ci-base-development-system repository")

## Sistem-Informasi-RT

---

Sistem-Informasi-RT is a web app to management civil registration in a RT(Rukun Tetangga) area. This web has 3 feature, first manage civil data based on KK(kartu keluarga), second manage civil data based on KTP(Kartu Tanda Penduduk), and last feature to search civil data for others(not officer, like a regular civil).
![First Post](/images/first-post.png "First Post")
This web app was built with codeigniter 3.x.x framework, AdminLTE as a dashboard template(officer area), and interfaces for non-officer area [Goto repository](https://github.com/aziyan99/Sistem-Informasi-RT "sistem-informasi-rt repository")
