## Sistem-Informasi-RT

---

Sistem-Informasi-RT is a web app to management civil registration in a RT(Rukun Tetangga) area. This web has 3 feature, first manage civil data based on KK(kartu keluarga), second manage civil data based on KTP(Kartu Tanda Penduduk), and last feature to search civil data for others(not officer, like a regular civil).
![ci-rt](/images/portfolio/ci-rt-home.png "CI RT")
![ci-rt](/images/portfolio/ci-rt-dashboard.png "CI RT")
![ci-rt](/images/portfolio/ci-rt-penduduk.png "CI RT")
This web app was built with codeigniter 3.x.x framework, AdminLTE as a dashboard template(officer area), and interfaces for non-officer area [Goto repository](https://github.com/aziyan99/Sistem-Informasi-RT "sistem-informasi-rt repository")

## HMTI Election

---

HMTI Election is a web app for _Pemilihan Ketua Himpunan Mahasiswa Teknik Informatika_ in our University, for 2020-2021 priod, but in the end, we not use it for the election, but we use google form :D for avoid any fraud. This web app divided with 3 part, First the restful-api for handle the election (Built with lumen framework), second is a admin or dashboard area (built with angular framework), and the last the election area (also built with angular 😁). For deploy the the app, I use heroku to deploy the restful-api, and github pages as client app (admin or dashboard area)
![hmti-ecletion](/images/portfolio/pemilihan-hima-restapi.JPG "hmti-ecletion")
